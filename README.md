## How to use

Add the following line to your `package.json` file's `dependencies` section.
`"gogogab-models": "git+ssh://git@github.com:Gogogab/gogogab-models.git#master"`

```javascript
var models = require('gogogab-models');
//...
var Gab = models.Gab;
```
